Apache Celix

Apache Celix is an implementation of the OSGi specification adapted to C and C++. It is a provides a framework to develop (dynamic) modular software applications using component and/or service-oriented programming. Apache Celix is primarily developed in C and adds an additional abstraction, in the form of a library, to support for C++.
